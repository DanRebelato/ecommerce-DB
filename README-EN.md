E-commerce Database
A data modeling and analysis project simulating a complete retail sales system.

🎯 Objective
This project demonstrates the design of a relational database structure and the generation of business insights through data analysis.

📊 Sample Insights
Total Revenue: Tracking overall sales performance.

Best-Selling Products: Identifying top-performing items by volume.

Top Customers: Segmenting clients with the highest purchase volume.

Average Order Value (AOV): Calculating the mean value per transaction.

🧩 Features
Automated Inventory Management: Real-time stock control implemented via SQL triggers.

Price History: Tracking price fluctuations over time for auditing and analysis.

Relational Mapping: Robust architecture connecting Orders, Customers, and Products.

⚙️ How to Run
To set up the environment and run the analyses, execute the scripts in the following order:

01_schema.sql: Creates the database structure and tables.

02_inserts.sql: Populates the database with sample data.

04_triggers.sql: Sets up automation rules and business logic.

03_queries.sql: Runs the analytical queries to extract insights.
